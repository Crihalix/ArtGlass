О компании Art Glass

В течение года после выхода Windows 10 будет бесплатной для пользователей предыдущих версий. Коснется это и обладателей пиратских версий, которые смогут бесплатно обновиться до Windows 10. С целью привлечь максимальное количество пользователей к выходящей летом новой платформе Windows 10 корпорация Microsoft пойдёт на беспрецедентные меры. Бесплатные версии новой операционной системы будут предоставляться всем владельцам ПК на базе Windows 7 или 8.1, включая тех, кто использует пиратские версии. “Мы модернизируем все отвечающие системным требованиям ПК до Windows 10, невзирая на наличие лицензии”, — сказал Терри Майерсон (Terry Myerson), вице-президент Microsoft, отвечающий за операционные системы. “План — воссоединиться с сотнями миллионов пользователей Windows в Китае”.

Сразу после конференции официальный представитель Microsoft подтвердил, что бесплатно обновить систему вне зависимости от наличия лицензии смогут пользователи по всему миру – по крайней мере, в первый год после релиза. В январе Microsoft объявила о планах предоставить бесплатную возможность перехода на новую операционную систему Windows 10 владельцам ПК с лицензионными версиями Windows 7 и 8.1. Новый план компании куда более радикален: он предполагает предоставлять лицензию на Windows 10 в том числе пользователям “пиратских” версий операционных систем в мировом масштабе, чего раньше в Microsoft не делали фактически никогда. Маркетолог Microsoft Крис Капоселла (Chris Caposella) на конференции Microsoft Convergence показал слайд-иллюстрацию трех ведущих экосистем на рынке с ключевыми активами каждой из них.
